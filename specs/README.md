# Public Implementation Profiles

These documents describe the preview interoperability and security direction for Agent Capability Middleware. They are implementation profiles built on existing web and agent standards, not a proposal for a new transport protocol.

- [Permission manifest](permission-manifest.md)
- [Capability token](capability-token.md)
- [Audit log](audit-log.md)
- [x402 policy binding](x402-policy-binding.md)

Draft profiles may change before `1.0`. The non-production reference server demonstrates grants and revocation but does not implement every future profile.
